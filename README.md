Installing python-grpc-prometheus
==================

Installing `python-grpc-prometheus` from the main channel can be achieved by:

```
conda install python-grpc-prometheus
```

It is possible to list all of the versions of `python-grpc-prometheus` available on your platform with `conda`:

```
conda search python-grpc-prometheus
```
